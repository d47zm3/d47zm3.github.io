Update mutli-site primary URL

In order to update main blog URL update these tables in database:

wp_options: change “siteurl” and “home” to the new URL (http://blog/)
wp_sitemeta: there’s a “siteurl” field you’ll need to change here
wp_site: change the only value here to your new domain
wp_blogs: each subsite will have an entry here, you’ll need to update the domain column for each row

Also, update wp-config.php file:

define( 'DOMAIN_CURRENT_SITE', 'newblog.com' );

Wordpress, TLS and Kubernetes

Since when using (default) ingress you don’t have an option to include custom TLS config, you need to include these lines in your wp-config.php and TLS should work properly.

if ($_SERVER['HTTP_X_FORWARDED_PROTO'] == 'https') $_SERVER['HTTPS']='on';
if (isset($_SERVER['HTTP_X_FORWARDED_HOST'])) $_SERVER['HTTP_HOST'] = $_SERVER['HTTP_X_FORWARDED_HOST'];
