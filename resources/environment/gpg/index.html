GPG

Set up GPG on computer to encrypt data & sign commits etc.

MacOS Guide

~> brew install gnupg pinentry-mac
~> cat ~/.gnupg/gpg-agent.conf
# Connects gpg-agent to the OSX keychain via the brew-installed
# pinentry program from GPGtools. This is the OSX 'magic sauce',
# allowing the gpg key's passphrase to be stored in the login
# keychain, enabling automatic key signing.
pinentry-program /usr/local/bin/pinentry-mac

~> cat ~/.gnupg/gpg.conf
# Uncomment within config (or add this line)
use-agent
no-tty
# This silences the "you need a passphrase" message once the passphrase handling is all set.
# Use at your own discretion - may prevent the successful interactive use of some operations.
# It is working fine for my use cases though.
batch

For various commands, check cheatsheet here.

Sample commands to encrypt & decrypt files for use in script (batch mode):

# encrypt
gpg --output /mnt/offsite/${database}_${backup_date}.sql.gpg --passphrase="${passphrase}" --batch --symmetric /mnt/offsite/${database}_${backup_date}.sql
# decrypt
gpg --output /mnt/offsite/${database}_${backup_date}.sql --decrypt --passphrase="${passphrase}" --batch --symmetric /mnt/offsite/${database}_${backup_date}.sql.gpg
