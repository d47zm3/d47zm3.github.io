DVNA

As a project for upcoming company’s internal hackday I chose to make workshops for secure coding in JavaScript, based on OWASP TOP 10 and Damn Vulnerable NodeJS Application. While I’m no coder by myself, I thought I might prepare as much I can from security perspective and use some feeback from more knowledgeable collegue in terms of code itself. The idea came when I stumbled upon “solution” book to DVNA itself from AppSeco. While I could just tell developers, “hey, check out this link!”, that would be best way for them to ignore this, hands-on workshop from “security” however, sounds a lot better. After I’m done with these, I will translate slides/possible script and put them here hopefully. That should count as mini-project…
